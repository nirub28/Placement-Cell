## Placement Cell App

This is a web application for managing student profiles and interviews in a placement cell.

## Project setup on local system

1. Clone the repository:

   ```shell
   git clone https://github.com/nirub28/Placement-Cell

2.Install the dependencies:

 using `npm install ${name}` download dependencies used in package.json file 

3.Set up the necessary configuration files and database.
   
   - we are using mongodb, hence download mongoDB on your system.

4. Start the application: `npm start`

5. Access the application in your web browser: `http://localhost:8000`

## Folder Structure

- `config`: Contains configuration files and modules.
- `controllers`: Contains the application's controllers or route handlers.
- `models`: Contains the Mongoose models for the application's data.
- `assets`: Contains static assets like CSS and JavaScript files.
- `routes`: Contains the application's route definitions.
- `views`: Contains the EJS templates for rendering the application's views.
- `index.js`: The main entry point of the application where the server is started and configured.

## Features
This application can be used for

1. Adding students to the placement app and tracking their data
2. Can create a interview and add students and update results of each student
3. Can also update final status of the student
4. We can download data of all students to make required reports
